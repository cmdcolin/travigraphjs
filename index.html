travigraph.js

Plot the travis-CI build times of a repo using the travis-CI API

Help

Usage

Simply enter the user/org and repository name to look up on the Travis-CI platform, along with the start and end build numbers (use 0 and a semi-large number to get whole history, total build numbers is not known beforehand)

Troubleshooting

If you get errors, make sure to use capitalization-correct names of github org and repo name (github is flexible on capitalization, travis-CI is not)

Note that large outliers are removed automatically (stalled builds, etc)
